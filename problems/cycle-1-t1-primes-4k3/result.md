# Result

Headline: An elementary classical Euclid-style argument proves infinitely many primes congruent to 3 modulo 4.

For every integer N >= 1, there exists a prime p > N with p congruent to 3 modulo 4.

Proof: see `proofs/primes-4k3.fml`. The construction assumes all primes 3 modulo 4 are bounded by N, multiplies the bounded list, forms M = 4Q - 1, and extracts a new prime divisor congruent to 3 modulo 4 that divides neither the product nor 1.

Significance: This establishes the T1 target exactly and is corpus-eligible as a self-contained elementary theorem.
