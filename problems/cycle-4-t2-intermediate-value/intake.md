# Intake

## A. Formal Restatement
For all real a < b and all continuous f : [a,b] -> R, if f(a) <= 0 <= f(b) or f(b) <= 0 <= f(a), then there exists c in [a,b] such that f(c)=0.

## B. Classification
MSC 2020: 26A06. Type: proof.

## C. Success Criteria
Primary target: full rigorous proof with endpoint-zero cases included.
