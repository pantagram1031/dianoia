## Interpretation

The exact rank-to-rank vectors are a useful invariant for this restricted
width-3, height-4, rank-shape `2,2,2,1` class. The coarse counts already
isolated most near-boundary behavior, but they could not tell whether the same
number of adjacent and skip covers were arranged in the same ranks. Vector mode
adds that missing placement information.

The next lemma target should not mention the eight processed buckets by name.
It should use the vector features as hypotheses, then prove a uniform lower
bound for each vector class. A promising shape is:

```text
For rank-shape 2,2,2,1 width-3 posets, if the cover matrix has a vector
feature class outside the eight processed near-boundary vector classes, then
some incomparable pair has lower orientation probability at least 13/32.
```

That statement is still only a candidate. It needs a human-readable proof
scheme, not just finite data.
