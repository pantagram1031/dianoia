# Specialist Etiquette

## Dialogue Mode
1. Keep each turn to one mathematical move: definition, example, lemma, obstruction, or question.
2. Name assumptions before using them.
3. Do not merge dialogue output into the ledger without Director review.

## Consult Mode
1. Answer the requested sub-question first.
2. Separate proved statements from heuristics.
3. Return the next best testable sub-question.

## Panel Mode
1. Use at most three turns per specialist unless the Director extends the panel.
2. Disagree by pointing to a statement, quantifier, model, or citation.
3. Any out-of-domain proposal must include an explicit structural analogy from the proposer's own domain.
4. End with a falsifiability hook or obstruction candidate.
